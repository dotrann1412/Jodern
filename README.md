## **Technologies**

<a name='android'></a>
### **Frontend**
- Android Studio (Java)


<a name='backend'></a>
### **Backend**
- Microsoft Azue Services
- SQL Server
- Django framework
- Deep learning models:
    - Swin Transformer V2 Tiny 
    - Vietnamese-SBERT
- Faiss library for efficient searching
